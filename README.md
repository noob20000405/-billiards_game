# billiards_game
final java project using the 8-balls rules
## basic rules
- two player
- seven solid balls numbered 1 - 7, seven striped balls numbered 9 - 15, an 8 ball, and a cue ball(white ball)
- 一人击1 - 7号球, 另一人击9 - 15号, 完成该任务后击打8号球入袋者获胜
